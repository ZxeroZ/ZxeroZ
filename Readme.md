## 🧠 About Me

I build backends with actual logic (not just basic CRUDs), throw together apps when needed, and mess around with Android when the stock software gets annoying.

- ⚙️ **Backend:** Writing APIs that handle async processes and strict validations.
- 📱 **Mobile/Android:** Creating LSPosed hooks, or using Flutter to build utility apps (like package tracking tools) just to get things done.
- 🎯 **Philosophy:** If a feature doesn't exist, just build it (or hook it).
